

 ### 本项目是用python实现的字符统计，行统计以及词频统计分析

 ### 项目分布
* src下为具体工程文件
* Detail.py为功能函数实现文件
* WordCount.py为main函数文件
* codestyle.md为代码规范

 ### 运行方式
* 在命令行输入 python WordCount.py inputfile outputfile
* inputfile和outputfile均使用相对路径
