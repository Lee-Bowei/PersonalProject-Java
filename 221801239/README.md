## 项目介绍
### 1.如何运行
在项目所在文件夹底下打开cmd，输入java 输入文件的文件名 输出文件的文件名    ,这样就可以运行项目了。
### 2.功能简介
本项目主要功能是统计文件中字符数、有效行数(只包含空白字符的为无效行)、单词数量(单词定义为至少四个字母打头，后面跟上任意个数字或字母)、单词频数前十的单词以及出现的次数。
