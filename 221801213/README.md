
>## README

- 项目：文本词频统计

- 功能：对一个文本的字符数、行数、单词数统计以及统计显示出现频率最高的10个单词

- 如何运行：通过在项目目录下命令行运行java WordCount input.txt output.txt，其中input为要统计的文本，output存放结果
