# 项目描述

## 如何运行
>在命名行编译时: `javac -encoding UTF-8 WordCount.java` <br>
>在命名行运行：`java WordCount input.txt output.txt` <br>

## 功能简介
>本项目用于英文文本的词频统计，可以统计出总字符数、总单词数，以及频率top10的单词
