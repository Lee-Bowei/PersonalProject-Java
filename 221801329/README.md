# PersonalProject-Java
## 项目描述
本项目用于英文文本的词频统计，可以统计出总字符数、总单词数，以及频率top10的单词
## 运行方式
`javac -encoding UTF-8 WordCount.java`

`java WordCount input.txt output.txt`

JVM内存支持的话可以运行100,000,000行以上的数据
