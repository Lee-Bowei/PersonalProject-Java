#WordCount
## 1.如何运行
> 读入一个文件，通过按字符读取获得字符数，通过按行处理获取行数，并将每行读取到的字符串送去字符处理，处理后将合法的单词送入TreeMap，再利用比较器进行排序
![此处输入图片的描述][1]
## 2.功能简介

 - 获取文章中的字符数
 - 获取文章的行数
 - 获取文章中的合法单词数
 - 获取词频高的单词和频率
